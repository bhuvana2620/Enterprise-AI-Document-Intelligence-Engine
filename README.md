# Enterprise Document Intelligence Engine

A production-style RAG document intelligence platform using FastAPI, Streamlit, Pinecone, semantic embeddings, CrossEncoder reranking, OCR support, and Gemini/Grok LLM fallback.

## Features

- Streamlit frontend
- FastAPI backend
- Pinecone vector database
- BAAI/bge-small-en-v1.5 semantic embeddings
- BAAI/bge-reranker-base reranking
- OCR support for scanned PDFs
- Gemini and Grok LLM fallback
